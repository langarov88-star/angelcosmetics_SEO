# H1 Templates — angelcosmetics.bg

**Version:** v1.0
**Rule:** Всяка indexed страница има точно един H1. H1 ≠ SEO Title — може да се различава леко.

---

## Принципи

- H1 е видим на потребителя — трябва да звучи естествено
- H1 трябва да съдържа primary keyword, но не е нужно да е точно match
- H1 е по-дълъг и описателен от SEO Title
- Никога не повтаряй H1 на друга страница

---

## Templates по Page Type

### Home Page

```
{Brand Name} — {1-2 думи positioning}
```

Пример:
```
Angel Cosmetics — дерматологична козметика онлайн
```

---

### Main Category Page

```
{Category Name} — {short benefit modifier}
```

Примери:
```
Козметика за лице — хидратация, защита и сияние
Козметика за коса — подхранване, сила и блясък
Грижа за тяло — омекотяване и хидратация
Слънцезащита — SPF за лице и тяло
```

**Правило:** Modifier трябва да описва ефект, не продуктов тип. "хидратация, защита и сияние" — да. "кремове, серуми и маски" — не.

### Subcategory Page

```
{Subcategory Name} за {target} — {benefit}
```

Примери:
```
Серуми за лице — концентрирана грижа за всеки тип кожа
Хидратиращи кремове — дълготрайна хидратация за лице
Почистващи продукти за лице — нежно и ефективно почистване
Шампоани — грижа за скалп и коса
```

**Когато subcategory е тесна:** Може без modifier ако е self-explanatory:
```
Пилинги за лице
Маски за коса
Масла за лице
```

## H1 vs. SEO Title — Разлика

| | SEO Title | H1 |
|--|-----------|-----|
| **Цел** | Ранкиране + CTR в SERP | UX + keyword relevance на страницата |
| **Дължина** | 50-60 chars | По-свободна — 60-90 chars |
| **Съдържа** | Primary KW + Brand name | Primary KW + descriptive modifier |
| **Видимост** | В browser tab + Google | На страницата |

**Пример — Category Page:**
```
SEO Title: Серуми за лице | Онлайн Козметика | Angel Cosmetics
H1:        Серуми за лице — концентрирана грижа за всеки тип кожа
```

---

## Забранено в H1

- Keyword stuffing: "Серуми серум за лице хидратиращ серум" — **НЕ**
- Generic marketing: "Най-добрите продукти" без specificity — **избягвай**
- Дублиране на H1 от друга страница — **никога**
- H1 идентичен с SEO Title — **избягвай** (малка разлика е ОК)
- Caps lock или ALL CAPS — **НЕ**
- Специални символи (@, #, !, ?) — **НЕ**
