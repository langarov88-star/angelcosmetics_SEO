# Meta Description Templates — angelcosmetics.bg

## Правила

- Оптимална дължина: 150-160 символа
- Трябва да съдържа primary keyword (натурално)
- Трябва да има call-to-action
- Не е ranking factor — но влияе на CTR
- Уникална за всяка страница

---

## Templates по Page Type

### Home Page
```
Template:
  {Brand positioning sentence}. {N}+ продукта от топ дерматологични марки.
  Безплатна доставка над {X}лв. Поръчай онлайн.

Example:
  Онлайн козметика с продукти от La Roche-Posay, CeraVe, The Ordinary и още 50+
  марки. Безплатна доставка над 60лв. Поръчай бързо онлайн.
  [157 chars]
```

### Main Category Page
```
Template:
  Открий {N}+ {category} от водещи дерматологични брандове. {Benefit statement}.
```

## Елементи, Които НИКОГА не Включваме

```
❌ Keyword stuffing: "козметика за лице крем за лице серум за лице"
❌ Generic statements: "Добре дошли в нашия магазин"
❌ Promises без стойност: "Намери перфектния продукт за теб"
❌ Clickbait: "НЕ купувай преди да прочетеш това"
❌ Цени в meta description (сменят се, meta може да не се обнови)
❌ Duplicate meta descriptions (дори близки)
```
