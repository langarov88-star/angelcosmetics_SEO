# Problem / Concern Page Template — angelcosmetics.bg

## Кога се прилага

За всички `/skin-concerns/{slug}/` и `/hair-concerns/{slug}/` страници.

**Това е highest-leverage page type.** Приоритет в Phase 1 и Phase 2.

**INDEX условие:** ≥600 думи оригинално съдържание

## Пълна Структура на Страницата

### Блок 1: Problem Definition (задължителен)
```
Word count: 100-150 думи
Tone: Практично, не медицинско. Не диагностициращо.

Съдържание:
  - Какво означава {problem} (практична дефиниция)
  - Как изглежда и как се усеща
  - Защо е важно да се грижим правилно

Пример (суха кожа):
  "Суха кожа означава, че кожата произвежда по-малко себум от необходимото,
   което води до усещане за стягане, лющене и дискомфорт. За разлика от
   обезводнената кожа (която е временно състояние), сухата кожа е тип кожа —
   наследствена характеристика, която изисква постоянна грижа с правилните продукти."
```

### Блок 2: Причини (задължителен)
```
Format: Bullet list, 4-6 точки
Tone: Информативен, не катастрофичен

Примери за суха кожа:
  • Генетична предразположеност
  • Студен или сух климат
  • Прекалено гореща вода при миене
  • Агресивни почистващи продукти
  • Стареене (производството на себум намалява с възрастта)
  • Определени медикаменти или здравословни условия
```

### Блок 3: Ключови Съставки (задължителен)
```
Word count: 150-200 думи total за секцията
Format: Списък с mini-description + link към ingredient page

Структура за всяка съставка:
  **{Ingredient Name}** ([виж пълното ръководство →](/ingredients/{ingredient-slug/}))
  Кратко описание защо е ефективна за този проблем (2-3 изречения)

Примери за суха кожа:
  - Хиалуронова киселина → /ingredients/hyaluronic-acid/
  - Керамиди → /ingredients/ceramides/
  - Урея → /ingredients/urea/
  - Глицерин → /ingredients/glycerin/
```

### Блок 4: Препоръчана Рутина (задължителен)
```
Word count: 150-250 думи

### Сутрешна рутина
1. Стъпка — [продукт тип] (link към subcategory или конкретен продукт)
2. Стъпка — ...
3. SPF — [задължително за всяка сутрешна рутина]

### Вечерна рутина
1. Стъпка — [продукт тип]
2. ...
3. Нощен крем — [link]

Tone: Конкретен и actionable. Не generic.
```

### Блок 5: Препоръчани Продукти (задължителен)
```
Брой: 6-12 продукта
Format: Product cards с кратко обяснение ЗАЩО е препоръчан

НЕ е:
  - Automated product listing по category filter
  - Случаен selection без editorial обяснение

ДА е:
  - Curated editorial selection
  - Всеки продукт с 1-2 изречения защо е подходящ
  - Разнообразие по ценова категория
  - Разнообразие по бранд
```

### Блок 6: Препоръчани Серии (желателен)
```
Брой: 2-4 серии
Format: Series cards с описание
Links към series pages
```

### Блок 7: От Блога (желателен)
```
Брой: 2-3 articles
Format: Thumbnail + Title + Short excerpt
Links към /blog/{article-slug}/
```

### Блок 8: FAQ Block (задължителен)
```
Брой Q&A: 5-7
Schema: FAQPage JSON-LD
Format: Accordion

Типични въпроси за суха кожа:
  Q: Каква е разликата между суха и обезводнена кожа?
  Q: Може ли суха кожа да стане мазна с времето?
  Q: Как да изберем крем за суха кожа?
  Q: Трябва ли серум при суха кожа?
  Q: SPF при суха кожа — кой да изберем?
```

### Блок 9: Свързани Проблеми
```
Format: Links
Example за суха кожа:
  "Виж също: Грижа за чувствителна кожа | Обезводнена кожа | Анти-ейджинг"
```

---

## Cannibalization Check

Преди да пишеш — провери:
- Primary keyword на тази страница ≠ primary keyword на свързана category subcategory
- Ingredient pages таргетират "какво е X" — НЕ "продукти за X"
- Blog articles таргетират "кой е най-добрият X" — НЕ "грижа за X"
