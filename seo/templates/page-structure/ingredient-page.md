# Ingredient Page Template — angelcosmetics.bg

## Кога се прилага

За всички `/ingredients/{ingredient-slug}/` страници.

**INDEX условие:** ≥200 думи оригинален текст. Не дублира problem pages.

**КЛЮЧОВО разграничение:**
- Ingredient page = "Какво е X и как работи"
- Problem page = "Как да се грижим за Y кожа" (X е само споменат)

## Структура на Страницата

### Блок 1: Какво е (задължителен)
```
Word count: 80-120 думи

Съдържание:
  - INCI name (ако е различно от popular name)
  - Кратка дефиниция (не от Уикипедия)
  - Откъде идва / как се получава (ако е интересно)

Example (Ниацинамид):
  "Ниацинамидът е форма на витамин В3, широко използвана в съвременната козметика
   заради впечатляващия си профил на безопасност и multi-tasking ефекти върху кожата.
   Научно доказан в концентрации от 2% до 10%, той работи без да раздразва дори
   много чувствителна кожа."
```

### Блок 2: Как Работи (задължителен)
```
Word count: 100-150 думи

Съдържание:
  - Механизъм на действие (практично, не химична лекция)
  - Какви ефекти дава (конкретни, не generic)
  - Времеви хоризонт за резултати

НЕ е медицинска информация. НЕ твърди лечение на заболявания.

Example:
  "Ниацинамидът работи на няколко нива: регулира производството на себум (мазнини),
   намалява видимостта на порите, изветлява петна по кожата и подсилва кожната
   бариера. Видими резултати при редовна употреба: 4-8 седмици."
```

### Блок 3: За Кого е Подходящ (задължителен)
```
Format: Кратък текст + skin type tags с links

Example:
  Ниацинамидът е подходящ за почти всички типове кожа:
  ✅ [Мазна кожа] → /skin-concerns/oily-skin/
  ✅ [Акне кожа] → /skin-concerns/acne-prone-skin/
  ✅ [Комбинирана кожа] → /skin-concerns/combination-skin/
  ✅ [Чувствителна кожа] → /skin-concerns/sensitive-skin/
  ⚠️ Внимание при: Много висока концентрация (>10%) може да раздразни
```

### Блок 4: Как се Използва (задължителен)
```
Word count: 80-120 думи

Съдържание:
  - Препоръчана концентрация
  - AM / PM / Both
  - Стъпка в рутината (преди или след)
  - Честота

Example:
  Концентрация: 2-10% (5% е добър баланс за начинаещи)
  Кога: Сутрин И вечер
  Стъпка: След тонер, преди крем
  Честота: Ежедневно
```

### Блок 5: Комбинации
```
Format: Две колони — "Комбинира се добре с" и "Внимание при комбиниране"

Добри комбинации:
  ✅ Хиалуронова киселина → /ingredients/hyaluronic-acid/
  ✅ Витамин С (нов мит — комбинацията е OK)
  ✅ Пептиди → /ingredients/peptides/
  ✅ SPF

Внимание:
  ⚠️ Много висока концентрация ретинол + висок % ниацинамид → може да раздразни
     Решение: разделяй AM/PM
```

### Блок 6: Продукти с {Ingredient} в Angel Cosmetics
```
Брой: 4-8 curated продукта
Format: Product cards с концентрацията на съставката

НЕ е automated listing. Curated editorial избор.

Example:
  - The Ordinary Niacinamide 10% + Zinc 1% — high-concentration серум
  - Paula's Choice 10% Niacinamide Booster — за напреднали
  - CeraVe Retinol Serum — с ниацинамид в подкрепяща роля
```

### Блок 7: FAQ (задължителен)
```
Брой: 4-6 Q&A
Schema: FAQPage JSON-LD

Примерни въпроси за ниацинамид:
  Q: Каква концентрация ниацинамид е оптимална за начинаещи?
  Q: Може ли да се комбинира ниацинамид с витамин С?
  Q: При каква кожа не е подходящ ниацинамидът?
  Q: Кога ще видя резултати от ниацинамид?
  Q: Ниацинамид сутрин или вечер?
```
