# Brand List — angelcosmetics.bg

## Правила за Brand Pages

- Brand page = INDEX само ако има editorial content (250+ думи)
- Всяка серия с ≥3 products получава собствена Series page
- Brand slug = lowercase, дефиси, без специални символи

## Как се добавя нов бранд

```
1. Добави бранда в тази таблица
2. Дефинирай URL slug
3. Идентифицирай серии с ≥3 products
4. Отвори GitHub Issue: "New brand page: {brand}"
5. Създай content brief от template
6. Добави в /seo/taxonomy/keyword-map.md
```
