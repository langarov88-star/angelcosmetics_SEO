# Prompt: Meta Description Suggestions

**Version:** v1.0
**Use on:** Всички page types при нужда от meta description варианти
**Output:** 3 варианта на meta description

---

## ПРАВИЛА ПРЕДИ УПОТРЕБА

- [ ] Знам primary keyword на страницата
- [ ] Имам H1 и page type
- [ ] Ще избера ЕДИН от 3-те варианта (може да редактирам)

---

## PROMPT TEMPLATE

```
You are an SEO specialist for Angel Cosmetics (angelcosmetics.bg), a Bulgarian
cosmetics e-commerce store.

TASK: Write 3 variations of a meta description for the following page.

PAGE TYPE: [Category / Brand / Product / Problem / Ingredient / Blog / Routine]
H1 OF THE PAGE: [paste H1]
PRIMARY KEYWORD: [keyword]
KEY PAGE CONTENT: [2-3 bullet points describing what's on the page]
LANGUAGE: Bulgarian

STRICT RULES:
1. Length: Exactly 150-160 characters each (count carefully)
2. Each variation must include the primary keyword naturally
3. Include a clear call-to-action (e.g., "Поръчай", "Открий", "Виж рутината", "Прочети")
4. No clickbait, no marketing superlatives ("най-добрият", "перфектният")
5. No price claims (prices change, meta descriptions are cached)
6. Each variation should have a different angle or emphasis
7. Write in Bulgarian

OUTPUT FORMAT:
Variant 1 (focus: [describe the angle]):
[meta description] ([character count] chars)

Variant 2 (focus: [describe the angle]):
[meta description] ([character count] chars)

Variant 3 (focus: [describe the angle]):
[meta description] ([character count] chars)
```

---

## REVIEW CHECKLIST

- [ ] Всеки вариант е 150-160 символа
- [ ] Primary keyword е включен натурално
- [ ] Има call-to-action
- [ ] Тонът е практичен, не promotional
- [ ] Избери ЕДИН вариант (или редактирай)
- [ ] Запиши в content brief или директно в Summer Cart
