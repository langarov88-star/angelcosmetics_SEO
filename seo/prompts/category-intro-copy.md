# Prompt: Category Intro Copy

**Version:** v1.0
**Use on:** Category pages, Subcategory pages с необходимост от editorial intro
**Output:** Draft intro copy (150-250 думи)

---

## ВАЖНО

AI draft = структура и keyword cues. НЕ е final text.
Content editor трябва да rewrite значително преди публикуване.

---

## PROMPT TEMPLATE

```
You are a senior content writer for Angel Cosmetics (angelcosmetics.bg),
a Bulgarian cosmetics e-commerce store specializing in dermatological skincare.

TASK: Write an editorial intro text for a category page.

CATEGORY NAME: [e.g., "Серуми за лице"]
URL: [e.g., /face-care/serums/]
PRIMARY KEYWORD: [e.g., "серум за лице"]
SECONDARY KEYWORDS: [e.g., "хидратиращ серум, серуми за лице, anti-aging серум"]
SUBCATEGORIES ON THIS PAGE: [list them, e.g., Хидратиращи, Anti-aging, Brightening, За акне]
TOP BRANDS IN CATEGORY: [e.g., La Roche-Posay, CeraVe, The Ordinary, Vichy]
MAIN SKIN CONCERNS THIS CATEGORY ADDRESSES: [e.g., суха кожа, акне, бръчки, пигментация]
LANGUAGE: Bulgarian

STRICT RULES:
1. Length: 180-220 words
2. Do NOT start with the category name ("Серумите са...")
3. Do NOT use generic openers ("Добре дошли в нашата...")
4. Include primary keyword in the first 50 words (naturally)
5. Mention 2-3 subcategory types naturally
6. Mention 1-2 brand names (naturally, not promotional)
7. Include practical advice on HOW TO CHOOSE from this category
8. No medical claims
9. No superlatives ("най-добрият", "революционен")
10. Write in Bulgarian
11. Practical, informative tone — not salesy

OUTPUT FORMAT:
[Intro text — 180-220 words, single block, no headers]
```

---

## POST-GENERATION WORKFLOW

1. Провери дали primary keyword е в първите 50 думи
2. Провери за generic phrases — изтрий или rewrite
3. Content editor rewrite — добави brand voice и specifics
4. SEO Manager проверява keyword usage
5. Публикувай само approved version

---

## НЕ ТРЯБВА да съдържа (ако го виждаш в AI output → rewrite)

- "Открийте перфектния продукт"
- "Нашите продукти са най-добрите"
- "Специално за вас"
- Generic skin type descriptions без конкретика
- Медицински claims
